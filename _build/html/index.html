Discord4J¶

Discord4J is an unofficial Java library that helps you access Discord’s API through REST and WebSockets.

Discord4J makes it easy for you to write a bot. It nearly has full api support. (See Discord API Comparison)

Feel free to join us in the Discord API chat.

You can find the source code on Github

Important¶

It’s recommended that you always use the latest version available. You can find the latest version in the Discord Chat & on Discord4J’s Github page

Please report any bugs you find to the Issue Tracker or to our Discord Chat

This Documentation is still under development. If this site cant help you, please use the Discord Chat
